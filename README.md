# ArbiGrow - Investment Platform

ArbiGrow is a structured digital investment platform currently under development. This is our Coming Soon Preview page with a live countdown timer and AI chat assistant.

## Features
- Live countdown timer showing time until official launch
- AI chat assistant with multilingual support (English/Bangla)
- Responsive design with hero background image
- Quick questions panel for first-time users
- Google Form integration for admin responses

## Available Scripts

In the project directory, you can run:

### `npm start`
Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.

## Deployment

### Vercel Deployment
This project is configured for easy deployment to Vercel:

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket)
2. Go to [Vercel Dashboard](https://vercel.com/dashboard)
3. Click "New Project"
4. Import your repository
5. Vercel will automatically detect the React app and deploy it

**Configuration:**
- Build Command: `npm run build`
- Output Directory: `build`
- Install Command: `npm install`

The project includes a `vercel.json` file with optimized settings for static site deployment.

### Manual Deployment
You can also deploy manually:
```bash
npm run build
# Then deploy the 'build' folder to any static hosting service
```
